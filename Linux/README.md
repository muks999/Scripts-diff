1. Системное администрирование и обслуживание
Скрипт auto-upgrade.sh: Автоматическое обновление системы и очистка (Bash)
Этот скрипт выполняет все этапы обновления системы и удаляет ненужные пакеты. Его можно добавить в cron для ежедневного/еженедельного выполнения.

Скрипт 2: Мониторинг места на диске и отправка уведомления
Скрипт проверяет заполненность корневого раздела и отправляет email-предупреждение, если место заканчивается.

2. Резервное копирование
Скрипт 3: Инкрементное резервное копирование с помощью rsync
Этот скрипт создает инкрементные бэкапы, используя жесткие ссылки, что экономит место. Похоже на работу "снимков" (snapshots).

3. Сетевые утилиты
Скрипт 4: Мониторинг доступности хоста (пинг)
Простой скрипт для проверки, отвечает ли важный сервер (например, шлюз или веб-сервер).

Скрипт 5: Блокировка подозрительных IP-адресов с помощью fail2ban и iptables
Это упрощенный пример, который можно использовать, если fail2ban слишком сложен для задачи.

4. Работа с пользователями и файлами
Скрипт 6: Массовое создание пользователей
Полезно для подготовки одинаковых учебных или рабочих окружений.

Скрипт 7: Поиск и удаление старых файлов (например, логов)

5. Развертывание и разработка
Скрипт 8: Простой скрипт для деплоя веб-приложения (Git + сборка)
Предполагается, что приложение хранится в Git-репозитории.




Как использовать эти скрипты?
Создайте файл: nano /usr/local/bin/my-script.sh

Вставьте код: Скопируйте код из примера.

Сделайте исполняемым: chmod +x /usr/local/bin/my-script.sh

Запустите: sudo /usr/local/bin/my-script.sh

Автоматизация с помощью Cron
Чтобы скрипты выполнялись по расписанию, используйте crontab -e.
